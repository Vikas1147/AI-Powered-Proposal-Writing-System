## Introduction
This repository contains the frontend for the AI-Powered Proposal Writing System. It provides an intuitive user interface for creating, editing, and managing proposals powered by AI assistance.

## Features
- **User Authentication:** Secure login, signup, and password reset.
- **Dashboard:**
  - View and manage proposals.
  - Status tracking (Draft, Submitted, Approved).
- **Proposal Editor:**
  - Rich-text editor with formatting options.
  - AI-generated content insertion.
  - Section management (add, reorder, remove).
- **Responsive Design:** Optimized for desktop and mobile devices.

## Technologies Used
- **Framework:** React
- **Styling:** TailwindCSS
- **Icons:** Lucide-React

## Installation
### Prerequisites
- Node.js (v16 or higher)
- NPM or Yarn

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Vikas1147/AI-Powered-Proposal-Writing-System
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open the application in your browser at `http://localhost:5317`.
